@erwinstone/input-touchspin

v1.0.3

Javascript touch friendly input spinner

Github

Features

  • Pure javascript (no jQuery)
  • CSS Framework Agnostic
  • Using native HTMLInputElement.stepUp() and HTMLInputElement.stepDown() methods
  • Lightweight (< 70 LOC)

Installation

Install with npm:

npm install @erwinstone/input-touchspin

Install from cdn:



			
Usage/Examples

			
Place js before </body> tag:

			
<body>
	...
	<script src="https://unpkg.com/@erwinstone/input-touchspin@1.0.3/dist/input-touchspin.min.js"></script>
</body>

			
* You can also use input-touchspin as javascript module:

			
<script src="yourscript.js" type="module"></script>

			
In yourscript.js file, import InputTouchspin (Change location according to your path):

			
import InputTouchspin from './path/to/input-touchspin.module.js'

			
Or you can use it directly in the html script as a module:

			
<script type="module">
	import InputTouchspin from './path/to/input-touchspin.module.js'
</script>
